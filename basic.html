📦 Docker Basic Commands

🔍 Check Docker Version
docker --version

Displays the installed Docker version.

📋 List Running Containers
docker ps

Shows currently running containers.

📦 List All Containers
docker ps -a

Lists all containers including stopped ones.

📥 Pull an Image
docker pull nginx

Downloads the nginx image from Docker Hub.

🚀 Run a Container
docker run -d --name webserver -p 8080:80 nginx

Runs an nginx container in detached mode with port 8080 mapped to port 80.

📂 Enter a Running Container
docker exec -it webserver bash

Opens an interactive shell in the container named webserver.

⏹ Stop a Container
docker stop webserver

Stops the running container.

▶️ Start a Stopped Container
docker start webserver

Restarts a previously stopped container.

🗑️ Remove a Container
docker rm webserver

Deletes the container named webserver.

🧹 Remove an Image
docker rmi nginx

Removes the downloaded nginx image.

📜 View Container Logs
docker logs webserver

Prints logs generated by the container.

🛠 Build an Image
docker build -t myapp .

Builds a Docker image from a Dockerfile in the current directory.

🏷 Tag and Push to Docker Hub
docker tag myapp yourdockerhub/myapp docker push yourdockerhub/myapp

Tags and pushes your image to your Docker Hub repository.
