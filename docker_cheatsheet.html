📘 Docker Cheat Sheet

📦 Basic Commands

CommandDescription
docker psList running containers
docker imagesList available images
docker run hello-worldRun a test container

🐳 Volumes & Networks

CommandDescription
docker volume create dataCreate named volume
docker network create webnetCreate custom network

🔁 Docker Compose

CommandDescription
docker-compose up -dStart services in background
docker-compose downStop and remove containers

⚡ Swarm Mode

CommandDescription
docker swarm initInitialize Docker Swarm
docker node lsList swarm nodes

🛠️ Services

CommandDescription
docker service create --name web -p 80:80 nginxCreate web service
docker service scale web=3Scale service to 3 replicas

🧠 Stacks

CommandDescription
docker stack deploy -c docker-compose.yml mystackDeploy stack using compose
docker stack rm mystackRemove a stack
