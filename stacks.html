What is a Docker Stack?

A Docker Stack is a collection of services that make up an application in a Docker Swarm. It allows you to deploy multiple services defined in a docker-compose.yml file using the docker stack deploy command.

Basic Commands

  • docker stack deploy -c docker-compose.yml mystack ‚Äî Deploy a new stack.
  • docker stack ls ‚Äî List all running stacks.
  • docker stack services mystack ‚Äî List services in a stack.
  • docker stack ps mystack ‚Äî List tasks (containers) in the stack.
  • docker stack rm mystack ‚Äî Remove a running stack.

Example: Web Application Stack

docker-compose.yml:

version: '3.8'
services:
  web:
    image: nginx
    ports:
      - "8080:80"

  app:
    image: node:18
    volumes:
      - ./app:/app
    working_dir: /app
    command: node index.js

  redis:
    image: redis:alpine

Deploy the stack:

docker stack deploy -c docker-compose.yml mystack

This example will:

  • Expose an Nginx web server on port 8080
  • Run a Node.js app from the ./app directory
  • Run a Redis service for caching
